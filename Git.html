Git Commands

  • git clone http://repository - clones repository
  • git status - shows git status
  • git add - add files
  • git commit -m "Message Here" - add message to read.md
  • git push - pushs files up to git
  • git pull - pulls files down from git

Mac Commands

  • cd - change directory
  • cd .. - go up a directory
  • ls - list directory contents
